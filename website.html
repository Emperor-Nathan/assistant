Custom Commands for Amazon's Alexa

Nathan Holmes-King
Siddharth Srinivasan



Design

This program allows the user to input a custom name for a command. Then, the user can use any name they want for that command.

The user can also have Alexa list all the names for a certain command.

Applications

If Alexa does not come in a user's native language, they can translate it into their language without having to wait for Amazon to do it. Also, Amazon can quickly translate without having to re-write any code.

Also, everyone can use shorthand as an alternative to saying long commands.
